方法/步骤1：进入到上传的文件的目录下，使用命令初始化本地仓库
git init

方法/步骤2：把本地文件添加到本地仓库暂存区,.的意思是把当前目录下所有的文件及子目录都添加管理，也可以把.换成相应的文件名
git add .
git add [filename]

方法/步骤3：把本地仓库暂存区的文件提交到本地仓库
git commit -m "update"

方法/步骤4：把本地仓库和远程仓库相关联，其中origin是远程仓库的别名，可以自己改变。
git remote add origin [url]
如：git remote add origin https://github.com/710146668/MIUITEST.git

方法/步骤5：如果远程仓库不为空，要把本地仓库和远程仓库做同步。否则可以省略此步骤，其中master为远程仓库的分支名。
git pull --rebase origin master

方法/步骤6：把本地仓库中的文件同步到远程仓库中。其中master为远程仓库的分支名。
git push -u origin master  或  git push origin master

方法/步骤7：最后可以使用命令查看状态
git status

强制更新覆盖远程仓库
git push -f origin master

存储帐号密码配置到本地
git config --global credential.helper store
